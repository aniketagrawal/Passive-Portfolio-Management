## Overview

Passive-Portfolio-Management is a tool designed to help users build a simple, low-maintenance fund that closely matches the performance of the NASDAQ-100 index. It selects 25 stocks out of 97 based on market data to create a balanced portfolio. The software uses mixed integer programming to find the best set of assets, aiming to keep the fund's performance similar across different market conditions.

This tool is useful if you want to follow the NASDAQ-100 index without buying all 100 stocks, reducing complexity and cost.

## 📚 How It Works

Passive-Portfolio-Management uses a mathematical approach to index replication:

- It compares hundreds of market scenarios to find a portfolio mix.
- The software applies mixed integer programming to ensure the portfolio matches the NASDAQ-100 as closely as possible.
- It focuses on "correlation-weighted similarity," which means the chosen stocks behave like the full index during varying market conditions.
- The result is a fund with fewer stocks but similar risk and return profiles to the original index.

This approach reduces transaction costs and simplifies management.
